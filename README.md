# Design and Simulation of a 2.45 GHz Circular Microstrip Patch Antenna using ANSYS HFSS

## Overview

This project presents the design and simulation of a **2.45 GHz Circular Microstrip Patch Antenna** using **ANSYS HFSS 2025 R2**. The antenna is designed for operation in the **2.4 GHz ISM Band**, which is widely used in Wi-Fi, Bluetooth, ZigBee, RFID, and IoT applications.

The project demonstrates the complete electromagnetic simulation workflow, including antenna modeling and analysis of radiation characteristics.

---

## Project Objectives

- Design a circular microstrip patch antenna operating at 2.45 GHz.
- Simulate the antenna using ANSYS HFSS.
- Analyze Terminal S-Parameters (S11).
- Generate Gain Polar Plot.
- Generate Directivity Polar Plot.
- Generate Far-Field Radiation Pattern.

---

## Software Used

- ANSYS Electronics Desktop 2025 R2
- HFSS (High Frequency Structure Simulator)
